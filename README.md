# SparseCore

> ⚠️ **Pre-alpha. Nothing here works yet.** If you arrived here expecting a library, come back after v0.1.

**SparseCore** is a dynamic sparse training engine for PyTorch, CPU-native, Apple Silicon first. It is under active construction.

Our goal: make training sparse-from-scratch the default choice for the open-source ML community, not an afterthought.

See `docs/PROJECT_OVERVIEW.md` for what we're building and why.
See `docs/LANDSCAPE.md` for how we fit into the existing sparse-ML ecosystem.
